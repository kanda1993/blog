exportされた会話ログファイルを記事用のlog.mdに変換してください。

ログファイル: $ARGUMENTS

## 手順

1. 指定されたログファイルを読み込む
2. 会話内容をMarkdown形式に整形する
   - User と Assistant の発言を見出しで区切る
   - コードブロックやリストはそのまま保持
   - 不要なメタ情報は除去
3. 対象の記事ディレクトリを確認する（どの記事に紐づけるか聞いてください）
4. `content/posts/YYYY/MM/DD/NNN/log.md` として保存する

## 出力形式

```markdown
# AI会話ログ

## User

（ユーザーの発言）

---

## Assistant

（アシスタントの発言）

---

（以下繰り返し）
```
